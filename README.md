# Forest Monitoring Finder

An interactive tool that helps restoration practitioners find the right monitoring approach for their project. Answer five questions about your goals, scale, ecosystem, budget, and technical capacity to get tailored recommendations from a curated database of monitoring tools, platforms, and advisory organizations.

## Local development

```bash
npm install
npm run dev
```

Open [http://localhost:5173](http://localhost:5173) in your browser.

## Build

```bash
npm run build
npm run preview   # preview the production build locally
```

## Deploy

This project is configured for deployment on [Vercel](https://vercel.com):

1. Push this repo to GitHub (see commands below).
2. Go to [vercel.com](https://vercel.com) → **Add New** → **Project**.
3. Import the GitHub repository.
4. Click **Deploy** — Vite is auto-detected, no configuration needed.

Subsequent pushes to `main` trigger automatic redeployments.
